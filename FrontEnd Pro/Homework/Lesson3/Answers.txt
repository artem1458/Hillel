1.  DOM существует для того, чтобы взаимодействовать с содержимым страницы.
    У DOM есть глобальный объект window.document, также у DOM, в отличии от BOM, есть собственный стандарт.

    BOM существует для того, чтобы взаимодействовать и получать информацю о браузере, текущем URL и тд.
    У BOM есть несколько объектов (location, navigator, screen и тд.) BOM является частью стандарта HTML5.

2.  С точки зрения исполнения кода, alert, в отличии от console.log,
    ждёт подтверждения от пользователя, тем самым блокирует дальнейшее выполнение кода, также как и confirm и prompt.
    Также, alert существует для взаимодействия с пользователем, а console зачастую используется для отладки, что требуется только разработчикам.

3.  Ошибки чтения - это синтаксические ошибки, например var a == b; или doSomething()); Эти ошибки возникают при опечатках.
    Ошибки выполнения - это любые другие ошибки, кроме синтаксических, например вызов несуществующего метода, обращение к несуществующей переменной и т.д.

4.  var 7e; не верна, имена переменных не могут начинаться с цифры. Также в имени переменной не должно быть символа "-". Имена переменных не должны быть
    зарезервированными словами языка.
    
5.  Любой примитив можно привести к булевуму 2-мя способами, Boolean(value) и !!value

6.  "23" + 20 + '' = "2320"
    //При сложении строк и чисел происходит преобразование числа к строке,
    а после - конкатенация строк. 

    "1" + (20 + 3) = "123"  
    //Сначала выполнится суммирование чисел в скобках, а после получившееся
    число преобразуется в строку, и произойдёт конкатенация.

    "a" / 5 = NaN   
    //При делении, умножении и вычитании строка преобразовывается к числу, так как
    строку "a", не получится привести к числу, вернётся NaN, NaN / 5 = NaN.

    ("str" || 5) > 0 || (10 && true) = true
    //Логический оператор || (или) возвращает первое значение, которое при превращении в булевое даёт true, если все ложные, возвращает последнее значение
    Логический оператор && (и) возвращает последнее значение, если все операнды true, если есть хоть один ложный, он возвращает его.
    Операторы сравнения >, < при сравнении строки с числом, преобразовуют строку в число, и сравнивают их
    ("str" || 5) вернёт "str"
    (10 && true) вернёт true
    "str" > 0 = NaN > 0 = false
    false || true = true
